Every php file that starts with UPPERCASE is a page that you can SEE, with lowercase you cannot
In `start.txt` link to start
In `admin` folder only admin files
In `auth` folder login and register forms
In `components` folder are components such as header, footer and so on 
In `css` styles
In `db` Everything that is somehow connected to database
In `developers` only developers files
In `images` for images
In `profile` profile stuff
In `purchase` purchasing system
In `review` review control
In `scripts` scripts in javaScript
